# Text Encryption Project

This project provides a simple user interface for encrypting and decrypting text using the Caesar cipher algorithm.

## Features

- Allows users to enter text to encrypt and specify a shift value.
- Encrypts the entered text using the Caesar cipher algorithm.
- Allows users to decrypt encrypted text by specifying the original shift value.

## Installation

1. Clone this repository to your local machine:

    ```
    git clone https://github.com/your-username/text-encryption.git
    ```

## Usage

1. Open the `index.html` file in a web browser.

2. Enter the text you want to encrypt in the input field labeled "Enter the text to encrypt".

3. Enter a shift value (a number between 1 and 25) in the input field labeled "Enter the shift value".

4. Click the "Encrypt" button to encrypt the entered text.

5. The encrypted text will be displayed in the input field labeled "Encrypted text".

6. To decrypt the encrypted text, enter the original shift value used for encryption in the input field labeled "Enter the shift value" and click the "Decrypt" button.

7. The decrypted text will be displayed in the input field labeled "Decrypted text".
